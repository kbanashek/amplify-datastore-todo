# Getting Started with Orion Task System

Welcome to the **Orion Task System** Storybook! This component library provides a comprehensive set of React Native components designed for health assessment and task management applications.

## üìö What's Inside

This library contains three main categories of components:

### 1. UI Components
Base design system components that form the foundation of the UI:
- **Button** - Primary action buttons with various styles
- **Card** - Container component for content grouping
- **TextField** - Text input with validation
- **DatePicker** - Date selection component
- **NumericInput** - Number input with unit support
- **LoadingSpinner** - Loading indicator
- **FieldLabel** - Form field labels with required indicator

### 2. Domain Components
Task and appointment management components:
- **TaskCard** - Display task information with actions
- **AppointmentCard** - Show appointment details
- **TaskFilters** - Filter tasks by various criteria
- **GroupedTasksView** - Display tasks grouped by date/time

### 3. Question Components
Health assessment question types for data collection:
- **SingleSelectQuestion** - Single choice selection
- **MultiSelectQuestion** - Multiple choice selection
- **TextQuestion** - Free-form text input
- **NumberQuestion** - Numeric value input
- **DateQuestion** - Date selection
- **BloodPressureQuestion** - Systolic/diastolic input
- **TemperatureQuestion** - Temperature measurement
- And 15+ more specialized types

## üöÄ Usage

### Installation

```bash
npm install @orion/task-system
# or
yarn add @orion/task-system
```

### Basic Usage

```typescript
import { Button, Card, TaskCard } from "@orion/task-system";

function MyApp() {
  return (
    <Card>
      <TaskCard
        task={myTask}
        onPress={handleTaskPress}
      />
      <Button
        title="Complete"
        onPress={handleComplete}
      />
    </Card>
  );
}
```

### With TypeScript

All components are fully typed with TypeScript:

```typescript
import type { Task, Question } from "@orion/task-system";
import { TaskCard, SingleSelectQuestion } from "@orion/task-system";

const task: Task = {
  id: "task-1",
  title: "Morning Medication",
  status: TaskStatus.OPEN,
  // ... other properties
};

const question: Question = {
  id: "q1",
  text: "Did you take your medication?",
  questionType: QuestionType.SINGLE_SELECT,
  // ... other properties
};
```

## üé® Design Principles

### Mobile-First
All components are designed for mobile devices first, with responsive design for tablets and desktops.

### Accessibility
Components follow accessibility best practices with proper ARIA labels and keyboard navigation.

### Theme Support
Components support light and dark themes through the theme system.

### Internationalization
All user-facing text is internationalized using i18next, supporting multiple languages.
